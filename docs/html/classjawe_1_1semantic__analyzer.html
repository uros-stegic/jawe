Detailed Description

template<typename... Analyzations>
class jawe::semantic_analyzer< Analyzations >

Compilation phase that it self is a phase based operation.

Like other phase based operations (jawe::phaser, jawe::optimizer), object of this class instantiates objects that does the real work.
