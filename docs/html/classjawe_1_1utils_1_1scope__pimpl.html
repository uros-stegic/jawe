Detailed Description

template<typename T = empty>
class jawe::utils::scope_pimpl< T >

Scope implementation when user needs to map certain type to variable's definition.

Intent of this class is to be used only by class jawe::utils::scope when user needs to map variable to some other object (e.g. llvm::AllocaInst).
